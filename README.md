此為 NCNU FPGA 期末專案

Authors: 112321073 112321079 112321080

Input/Output unit:

8x8 LED矩陣，用來顯示遊戲畫面。

七段顯示器，用來顯示分數

指撥開關，0為開始遊戲，1為reset遊戲

S1,S2,S3,S4按鈕，用來控制蛇的上下左右移動

功能說明:

使用FPGA上的S1,S2,S3,S4按鈕來控制蛇上下左右的移動，當達到8分時勝利，如果撞到牆壁則失敗。
